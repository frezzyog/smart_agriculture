System Ecosystem

The high-level data lifecycle of the Smart Agriculture platform, connecting physical sensors to cloud-based intelligence.

01

IoT Edge Layer

Industrial-grade sensors connected to ESP32 microcontrollers measure Soil Moisture, NPK levels, Temperature, and Humidity in real-time.

ESP32C++ / ArduinoI2C / ADC
02

Communication Bridge

Secure MQTT transmission sends sensor payloads to the Aedes Broker. This ensures ultra-low latency and reliable data delivery in remote areas.

Aedes BrokerMQTT 3.1.1JSON
03

Central Hub (Node.js)

The orchestration server handles data ingestion, authentication, and distributes real-time events via WebSockets to connected dashboards.

Node.jsSocket.ioPrisma ORM
04

AI Intelligence Layer

Python-based FastAPI service executes machine learning models. It performs anomaly detection and invokes Gemini AI for conversational farming advice.

FastAPIGemini 1.5 FlashScikit-Learn
05

Autonomous Actuation

Closing the loop: The AI triggers automated irrigation cycles or fertilizer pulses back through the MQTT bridge based on current soil health metrics.

Closed-Loop ControlRelay Modules

Error Recovery

• WiFi Loss: Device enters circular retry mode every 5s.
• MQTT Failure: Automatic reconnection without data drop.
• AI Timeout: System falls back to rule-based safety logic.
